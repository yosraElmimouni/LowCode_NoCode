# BookingApp - Guide Utilisateur

## À propos de BookingApp

**Objectif :** Application web complète de gestion de rendez-vous pour services de beauté, santé et soins.

**Accès :** Public - Aucune inscription préalable requise.

**Types de comptes :** Client ou Prestataire de services.

---

## Powered by Manus

**Stack technologique :**
- **Frontend :** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
- **Gestion d'état :** React Context API
- **Persistance :** localStorage pour données mockées
- **Routage :** Wouter (client-side routing)
- **Notifications :** Sonner (toast notifications)

**Déploiement :** Infrastructure auto-scaling avec CDN global et optimisation des performances.

---

## Utiliser BookingApp

### Pour les Clients

**Connexion rapide :** Cliquez sur "Client: Marie Dupont" pour accéder immédiatement.

#### 1. Parcourir les Services
- Accédez à "Services" dans la navigation
- Utilisez la **barre de recherche** pour trouver un service spécifique
- Filtrez par **catégorie** (Beauté, Santé, Soins)
- Consultez les détails : prix, durée, prestataire, avis clients

#### 2. Réserver un Service
- Cliquez sur "Réserver" sur la carte du service
- Sélectionnez une **date** dans le calendrier
- Choisissez un **créneau horaire** disponible
- Ajoutez des notes (optionnel)
- Confirmez la réservation

#### 3. Gérer vos Rendez-vous
- Consultez "Mes Rendez-vous" pour voir tous vos rendez-vous
- Filtrez par statut : Tous, À venir, Terminés
- **Modifiez** une réservation confirmée
- **Annulez** une réservation si nécessaire
- **Laissez un avis** après chaque rendez-vous complété

#### 4. Profil Personnel
- Cliquez sur "Profil" pour accéder à vos informations
- **Modifiez** votre nom, email, téléphone, adresse
- **Changez votre mot de passe** en toute sécurité
- Consultez votre photo de profil

### Pour les Prestataires

**Connexion rapide :** Cliquez sur "Prestataire: Sophie Bernard" pour accéder immédiatement.

#### 1. Tableau de Bord
- Consultez vos **statistiques** : nombre de services, réservations, rendez-vous d'aujourd'hui
- Visualisez votre **note moyenne** basée sur les avis clients
- Accédez rapidement à vos prochains rendez-vous

#### 2. Gérer vos Services
- Cliquez sur "Mes Services" pour voir votre catalogue
- **Créez un nouveau service** : nom, catégorie, prix, durée, description
- **Modifiez** un service existant
- **Supprimez** un service que vous ne proposez plus

#### 3. Gérer vos Réservations
- Consultez la liste de vos rendez-vous confirmés
- Visualisez les détails : client, date, heure, service
- Confirmez ou annulez les réservations

#### 4. Profil Professionnel
- Mettez à jour votre **nom commercial** et **bio**
- Gérez vos **spécialités**
- Modifiez vos coordonnées de contact
- Changez votre mot de passe

## Gérer votre Site

### Panneau de Gestion (Management UI)

Accédez aux paramètres via le bouton de menu en haut à droite.

**Panneaux disponibles :**

- **Preview :** Aperçu en direct de votre application avec persistance de connexion
- **Code :** Accédez à tous vos fichiers source et téléchargez le projet complet
- **Dashboard :** Statistiques de visite et visibilité du site
- **Database :** Consultez et modifiez les données (si backend activé)
- **Settings :** Configurez le titre, le logo, les domaines personnalisés
